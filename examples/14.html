Exemplo 14: order

Neste exemplo, queremos reorganizar os sapos em uma ordem específica. Usamos a propriedade order para mudar a ordem de exibição dos itens.

#frog1 {
  order: 2;
}
#frog2 {
  order: 1;
}
#frog3 {
  order: 3;
}

A propriedade order permite reordenar itens flexíveis sem modificar o HTML. Por padrão, todos os itens têm order: 0 e são exibidos na ordem do código HTML.

Valores menores de order são exibidos primeiro, enquanto valores maiores são exibidos depois.
