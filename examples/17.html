Exemplo 17: order & align-self

Neste exemplo, queremos mudar a ordem dos sapos e alinhar um deles na parte inferior do lago. Combinamos as propriedades order e align-self.

#frog1 {
  order: 2;
}
#frog2 {
  order: 1;
  align-self: flex-end;
}
#frog3 {
  order: 3;
}

Neste exemplo, combinamos duas propriedades para os itens flexíveis:

  • order: Reorganizamos a ordem horizontal dos sapos.
  • align-self: Movemos o segundo sapo para a parte inferior do lago.
