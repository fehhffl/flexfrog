Exemplo 18: flex-wrap: wrap

Neste exemplo, queremos que os sapos quebrem para a próxima linha quando não houver espaço suficiente. Usamos a propriedade flex-wrap com o valor wrap.

.pond {
  display: flex;
  flex-wrap: wrap;
}

A propriedade flex-wrap define se os itens flexíveis devem ser forçados a ficarem em uma única linha ou se podem quebrar em múltiplas linhas. Ela aceita os seguintes valores:

  • nowrap: Todos os itens ficam em uma única linha (padrão).
  • wrap: Os itens quebram em múltiplas linhas, de cima para baixo.
  • wrap-reverse: Os itens quebram em múltiplas linhas, de baixo para cima.
