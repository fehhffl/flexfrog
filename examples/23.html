Exemplo 23: Combinação de Propriedades

Neste exemplo avançado, combinamos várias propriedades para criar um layout complexo. Usamos flex-direction: column-reverse, flex-wrap: wrap-reverse e justify-content: center.

.pond {
  display: flex;
  flex-direction: column-reverse;
  flex-wrap: wrap-reverse;
  justify-content: center;
}

Neste exemplo complexo, combinamos várias propriedades Flexbox:

  • flex-direction: column-reverse: Os itens são distribuídos verticalmente, de baixo para cima.
  • flex-wrap: wrap-reverse: Os itens quebram para a próxima coluna quando necessário, da direita para a esquerda.
  • justify-content: center: Os itens são alinhados centralmente ao longo do eixo principal (agora vertical).

Este exemplo mostra como as propriedades Flexbox podem ser combinadas para criar layouts sofisticados.
