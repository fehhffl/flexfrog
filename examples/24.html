Exemplo 24: Combinação Completa

Neste exemplo final, combinamos todas as propriedades Flexbox que aprendemos para criar um layout complexo e personalizado.

.pond {
  display: flex;
  flex-flow: column-reverse wrap-reverse;
  justify-content: center;
  align-content: space-between;
}

#frog1 {
  order: 1;
  align-self: flex-end;
}

#frog2 {
  order: 2;
}

#frog3 {
  order: 3;
  align-self: flex-start;
}

Neste exemplo complexo, utilizamos diversas propriedades Flexbox:

  • flex-flow: column-reverse wrap-reverse: Propriedade abreviada que combina flex-direction e flex-wrap.
  • justify-content: center: Centraliza os itens no eixo principal.
  • align-content: space-between: Distribui as linhas igualmente no eixo transversal.
  • order: Altera a ordem de exibição dos elementos.
  • align-self: Permite o alinhamento individual de itens no eixo transversal.

Parabéns por chegar ao final dos exemplos de Flexbox! Agora você tem uma compreensão completa de como usar o Flexbox para criar layouts modernos e responsivos.
