Exemplo 8: flex-direction: row-reverse

Neste exemplo, queremos inverter a ordem dos sapos horizontalmente. Usamos a propriedade flex-direction com o valor row-reverse.

.pond {
  display: flex;
  flex-direction: row-reverse;
}

A propriedade flex-direction define a direção dos itens flexíveis no container e aceita os seguintes valores:

  • row: Itens são alinhados horizontalmente da esquerda para a direita (padrão).
  • row-reverse: Itens são alinhados horizontalmente da direita para a esquerda.
  • column: Itens são alinhados verticalmente de cima para baixo.
  • column-reverse: Itens são alinhados verticalmente de baixo para cima.
